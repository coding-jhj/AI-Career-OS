# Sprint 1 Plan - Core Development

## Goal

Create a runnable web application foundation for AI Career OS.

## Deliverables

1. Monorepo foundation
2. Next.js web application
3. Tailwind CSS styling foundation
4. Base dashboard shell
5. Supabase-ready configuration

## Acceptance Criteria

- The repository has a stable monorepo structure.
- `pnpm install` can install dependencies.
- `pnpm dev` starts the web app from the root.
- The dashboard route renders without backend dependency.
- Project status and task board are updated.

## Scope Control

Sprint 1 focuses on foundation only. AI features, job crawling, and automation are deferred to later sprints.
