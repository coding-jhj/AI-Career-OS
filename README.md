# AI Career OS

AI Career OS is a career operating system for developers who want to manage resumes, portfolios, GitHub activity, job applications, and AI-powered career coaching in one place.

## Current Status

- Current sprint: Sprint 1 - Core Development
- Current focus: Runnable Next.js web app foundation
- Repository: https://github.com/coding-jhj/AI-Career-OS

## Tech Stack

- Monorepo: pnpm workspace
- Web: Next.js, TypeScript, Tailwind CSS
- UI: shadcn/ui style component structure
- Backend: Supabase-ready
- Deployment: Vercel-ready

## Project Structure

```text
AI-Career-OS/
├── apps/
│   └── web/
├── docs/
├── packages/
├── supabase/
└── .github/
```

## Getting Started

```bash
pnpm install
pnpm dev
```

The web app runs from `apps/web`.
